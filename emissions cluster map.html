Emissions Cluster Map

This map is a visual representation of the places we have consistently found methane emissions in the GTA. Clusters are computed based on the entire archive of data from the GTA Urban Emissions Project's bike surveys.

A cluster not near a downward triangle may indicate an unknown (to us) source of methane emissions. If you live in the area and/or have any information about what the source of a cluster might be, please submit a cause report.

The "Show/Hide Cluster Centres" button toggles the centres of the clusters, which helps with visualizing where the clusters may be originating from.

Map Information
One circle represents a high level of methane measured in that spot, and a larger circle means a higher level of methane. A set of circles that are all the same colour represents a cluster. Clicking on each circle will show you the date that measurement was taken.
A dark purple icon represents a known source of methane emissions. Clicking on each icon will tell you the name of the source and the type of emitter it is.
